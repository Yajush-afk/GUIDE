# GUIDE

GUIDE is a Windows desktop overlay application that helps beginner laptop users learn how to use their computer through step-by-step visual guidance.

It is designed to feel like navigation for everyday computer tasks. Instead of only reading instructions or watching a video, the user receives guidance directly on top of their screen while they perform the action themselves.

GUIDE helps users understand what to do next by combining:

- on-screen overlay instructions
- highlighted regions and interface targets
- screen understanding
- local validation of user progress
- beginner-friendly hints
- optional AI-assisted explanations
- future voice guidance

The goal is to make common computer workflows easier for first-time and non-technical users. GUIDE does not take control of the computer for the user. It teaches by guiding, observing, validating, and explaining while the user remains in control.

GUIDE is focused on Windows laptops and desktops. It is intended for practical computer-learning scenarios such as opening applications, searching the web, using File Explorer, navigating basic system features, and completing common beginner workflows safely.

The long-term vision is an intelligent computer tutor that can appear when a user needs help, understand the current screen context, and guide the user through a task in a calm, reliable, and privacy-conscious way.
